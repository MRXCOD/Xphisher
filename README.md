#### INSTALL TOOL ON TERMUX
```python
$ apt update && apt upgrade
$ apt install python
$ pip install lolcat
$ pip install mechanize
$ pip install requests bs4
$ apt install git
$ git clone https://github.com/MRXCOD/Xphisher.git
```
#### RUN SCRIPT
```python
$ cd Xphisher
$ python ip2.py
```
